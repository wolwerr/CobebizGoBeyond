# CobebizGoBeyond

<h1>Desafio do Go Beyond Corebiz - BackEnd</h1>

Sistema para cadastro de pessoas e produtos, criado no curso Go Beyond BackEnd.

Foi utilizada a linguagem PHP com framework Laravel, Docker e Banco de Dados SQL.
